# Assignment 4 – FastAPI Cart System

This assignment implements a shopping cart system using FastAPI.

## Features
- Add items to cart
- View cart
- Remove items
- Checkout system
- Order history
- Error handling (out-of-stock, invalid product)
- Empty cart checkout protection

## Endpoints
- POST /cart/add
- GET /cart
- DELETE /cart/{product_id}
- POST /cart/checkout
- GET /orders
